Revisi tampilan halaman Inbox saya agar layout dan struktur UI-nya mengikuti referensi gambar pertama, tetapi JANGAN menggunakan warna dari referensi tersebut.

Gunakan tetap color system dari website saya saat ini:
- Dark navy background
- Glassmorphism card
- Cyan / blue accent
- Border halus
- Rounded corner
- Typography modern
- Nuansa premium SaaS dashboard

Fokus utama:
Buat tampilan Inbox menjadi lebih clean, simpel, rapi, dan tidak terlalu ramai seperti referensi pertama, tetapi tetap mempertahankan identitas visual website saya yang gelap dan modern.

Layout utama:
Gunakan 3 kolom seperti referensi:

1. Sidebar kiri
- Tampilkan logo/label “INBOX”
- Icon tambah (+) untuk membuat chat baru
- Icon search untuk mencari chat
- Menu folder:
  - All chats 45
  - My chats
  - Unassigned 32
  - Assigned 13
  - Resolved
- Menu aktif tetap memakai warna accent website saya, misalnya cyan/blue glow halus
- Tambahkan section CUSTOM VIEW dengan icon tambah (+)
- Bagian bawah berisi card:
  - Explore Inbox new look
  - Button “Switch to old inbox”
  - Button “Give feedback”
- Tambahkan icon collapse sidebar di pojok bawah

2. Kolom tengah / daftar chat
- Header besar: “All chats”
- Toolbar atas:
  - Dropdown sorting “Newest”
  - Icon checkmark untuk bulk action
  - Icon filter/corong
- List chat tampil clean seperti referensi:
  - Icon channel bulat, seperti WhatsApp atau Instagram
  - Nama pelanggan tebal
  - Preview pesan terakhir
  - Subtitle brand: “Johan Garage”
  - Waktu pesan di kanan atas
  - Status Assigned / Unassigned di kanan bawah
  - Badge merah jumlah pesan belum dibaca
- Setiap item chat diberi border-bottom halus
- Gunakan spacing lebih lega dan hierarchy teks yang jelas

3. Panel kanan / area detail chat
- Saat belum ada chat dipilih, tampilkan empty state di tengah:
  - Ilustrasi bubble chat sederhana
  - Text utama: “A chat will appear here”
  - Text kecil: “Select a chat to view customer messages.”
- Area kanan harus luas, bersih, dan tidak ramai
- Jika chat dipilih, tampilkan conversation history dan input balasan

Style visual:
- Jangan pakai light theme dari referensi
- Jangan pakai warna putih/lavender dari referensi
- Tetap gunakan dark theme website saya
- Ambil hanya struktur, spacing, dan kesederhanaan dari referensi
- Kurangi elemen yang terlalu ramai dari desain lama
- Pertahankan fitur yang sudah ada seperti:
  - AI Aktif
  - Assigned / Unassigned
  - unread badge
  - search
  - filter
  - sorting
  - resolve
  - take over
  - customer detail
  - custom view
  - switch old inbox
  - feedback

Jangan ubah logic backend, database, API, routing, atau fitur yang sudah berjalan.
Fokus hanya pada redesign UI/UX, layout, spacing, typography, visual hierarchy, dan konsistensi tampilan.

Target hasil:
Tampilan akhir harus terasa seperti versi dark premium dari referensi pertama:
lebih minimal, lebih bersih, lebih mudah dipakai, tetapi tetap sesuai branding website saya yang dark navy, cyan accent, dan modern SaaS.
